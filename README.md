Angular seed with RequireJS, Gulp, Sass


<h1>Install Dependencies:</h1>
<pre>
  <code>
  npm install
  </code>
</pre>
<pre>
  <code>
  bower install
  </code>
</pre>
<h1>Gulp Task</h1>

Create developer version:
<pre>
  <code>
  gulp build
  </code>
</pre>

Create version:
<pre>
  <code>
  gulp compile
  </code>
</pre>

<h3>Local server</h3>

Create developer version and start local server:
<pre>
  <code>
  gulp build-server
  </code>
</pre>

Create version and start local server:
<pre>
  <code>
  gulp compile-server
  </code>
</pre>
